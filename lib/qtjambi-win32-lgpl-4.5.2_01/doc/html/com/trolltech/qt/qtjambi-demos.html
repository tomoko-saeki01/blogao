Qt Jambi Demonstrations

This is the list of demonstrations in Qt Jambi's demos directory. These are larger and more complicated programs than the Qt Jambi Examples and are used to highlight certain features of Qt Jambi.

Painting

  • Vector Deformation demonstrates effects that are made possible with a vector-oriented paint engine.
  • Path Stroking shows Qt Jambi's built-in dash patterns and shows how custom patterns can be used to extend the range of available patterns.

Item Views

  • Spreadsheet demonstrates the use of a table view as a spreadsheet, using custom delegates to render each item according to the type of data it contains.

Rich Text

  • Text Edit shows Qt Jambi's rich text editing features and provides an environment for experimenting with them.

Main Window

  • Image Viewer shows how Qt Jambi can be used to make an application with standard GUI features such as menus, tool bars, and dock windows.

Graphics View

  • 40000 Chips shows how to visualize a huge scene with 40000 chip items using Graphics View.

Network

  • Simple HTTP Server shows how to create a http server by inheriting QTcpServer.

Phonon

  • Media Player demonstrates how the Phonon Module can be used to implement a basic media player application.
