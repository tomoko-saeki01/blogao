General Qt Requirements

Outlines the general requirements and dependencies needed to install Qt. This page describes the specific requirements of libraries and components on which Qt depends. For information about installing Qt, see the Installation page.

OpenSSL (version 0.9.7 or later)

Support for Secure Sockets Layer (SSL) communication is provided by the OpenSSL Toolkit, which must be obtained separately.
